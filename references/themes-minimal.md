# 极简白 · 配色方案

> 用法：复制一套主题的 `:root` 块，**整体替换** `template-minimal.html` 中的 `:root` 块。
> 每套主题包含完整变量（颜色 + 字体 + 间距 + 布局），直接粘贴即可，无需手动合并。

---

## 1. 石墨 (Graphite) — 默认

## 配色选择指南

| 用户身份/场景 | 推荐配色 |
|-------------|---------|
| 程序员、技术岗 | 石墨 或 靛蓝 |
| 创业者、产品经理 | 石墨 或 靛蓝 |
| 设计师、创意人 | 赭石 或 梅红 |
| 写作者 | 赭石 |
| 学生求职（技术向） | 石墨 |
| 学生求职（文商向） | 靛蓝 或 松绿 |
| 不确定 | 石墨（最安全） |
