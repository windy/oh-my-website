# 极简白 (Minimal)

纯文字排版，无图片，克制黑白灰 + 单一强调色。

## 适用身份
`coder` `professional` `founder` `student`

## 页面（8 页）
index, about, projects, blog, portfolio, product, resume, contact

## 图片处理
本模板不使用任何图片。生成时无需抠图或处理图片资源。

## 注意事项
- 无 `<body>` 身份 class，不调用 persona 特定样式
- 强调色在 `css/style.css` 的 `:root { --accent }` 中修改
- 页面最多，按用户身份删不需要的页面 + 清理导航
